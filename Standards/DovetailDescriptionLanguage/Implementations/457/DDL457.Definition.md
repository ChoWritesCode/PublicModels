- Implements: Dovetail Definition Language
- Implementation Type: Concrete
- Parameters (inherited)
    - Half Base Width: 4
    - Height: 5
    - Half Top Width: 7
- Additional parameters:
    - Driving
        - Label Total Thickness: Static, 5mm
        - Label Text Thickness: Static, 1mm
        - Dovetail Height: Dynamic
    - Driven
        - Label Base Thickness = Label Total Thickness - Label Text Thickness.
            - Derived: 4mm
        - Dovetail Lengths With Label Allowance = Dovetail Height - Label Total Thickness
            - This value is dynamically derived. The purpose to to allow enough clearance to push a label tag into the remaining space in the dovetail channel such that it snaps in securely.
